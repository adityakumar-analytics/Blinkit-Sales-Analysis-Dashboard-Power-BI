# 📊 Blinkit Sales Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** built using the Blinkit Grocery Sales dataset. The dashboard provides business insights into sales performance, outlet analysis, customer ratings, product categories, and inventory visibility.

The objective of this project is to practice data visualization, dashboard design, and business intelligence concepts using Microsoft Power BI.

## 📌 Key Insights

- Supermarket Type 1 generated the highest revenue.
- Regular Fat products contributed more sales than Low Fat products.
- Fruits and Snack Foods were among the highest-selling categories.
- Medium-sized outlets contributed the largest share of sales.
- Tier 3 outlets generated higher revenue compared to other outlet locations.
